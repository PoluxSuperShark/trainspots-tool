# 📦 Installation du projet

## 🧰 Prérequis
- Installer **XAMPP** : https://www.apachefriends.org/fr/index.html

## ⚙️ Installation

1. Télécharge et installe XAMPP.
2. Ouvre **XAMPP Control Panel** via le menu Démarrer.
3. Lance **Apache** et **MySQL**.
4. Ouvre le dossier web local :
   - `Windows + R`
   - Tape : `C:\xampp\htdocs`

5. Télécharge le projet :
   - Clique sur le bouton vert **Code**
   - Puis sur **Download ZIP**

6. Décompresse le ZIP.

7. Place le dossier du projet dans :
   - `C:\xampp\htdocs`

## 🚀 Lancement

1. Vérifie que **Apache** est bien démarré.
2. Ouvre ton navigateur.
3. Va sur : `localhost/index.php`
